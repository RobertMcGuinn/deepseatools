2 Getting data

2.1 Using ERDDAP

Getting data directly from ERDDAP (Environmental Resource Division Data Access Protocol) using ‘rerddap’ package.

First list all of the variable names from the National Database. To download an Excel version of the full data dictionary for the National Database following location: LINK

##### download information about the National Database for Deep Sea Corals and Sponges #####
info <- info(datasetid='deep_sea_corals', 
              url = "https://ecowatch.ncddc.noaa.gov/erddap/")

##### view head of variable names #####
